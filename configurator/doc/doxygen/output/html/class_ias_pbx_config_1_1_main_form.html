Detailed Description

The main program Form; contains everything else.

Contains The main menu, a split view area for browsing and viewing through the other forms, along with a tab area to switch between open forms. Also has a status bar at the bottom. Handles the creation and releasing of sub forms.

Todo:

Menus for handling db connections

Status bar interaction (message details (backtrace, full exception message))
