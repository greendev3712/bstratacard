Detailed Description

Handles Name Value Grid functionality through helper functions.

Each instance handles 1 grid, passed in through the constructor. A data object (

See also:
m_data[][]) is also passed in to initialize and setup the grid. A NameValueGrid handles an unlimited length list of Names and Value Controls (TextBox or ComboBox) and handles events for changing values.

add Controls (if needed) and events (which launch database actions) to the grid, which handle each row's Control.

TextBox and ComboBox Controls are supported. Enter Focus, Validating, and Validated events are supported. An anonymous delegate is created and added to the Grid for each Event. These delegates handle TextBox and ComboBox Controls in each row of the grid.

Parameters:
uniqueKeyValueHolderControl another control. check it for the currently selected unique key value.
uniqueKeyValueHolderControlCallback function to callback to pass back the new selected item in case this is a unique key selection control

CellValidating inline delegate function: handle CellValidating a new value set in a Control within a grid's DataGridViewCell
